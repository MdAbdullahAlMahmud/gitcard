# gitsnap

> Generate beautiful visual snapshot cards of any git repository.

## What it does

Run `npx gitsnap` on any git repository to produce a shareable 1200×630 image showing:

- Project name, description, version
- Language breakdown bar
- GitHub stats (stars, forks, issues, contributors)
- File count, repo size, total commits
- Recent commit history
- Topics / tags

## Install

```bash
# No install needed
npx gitsnap

# Or install globally
npm install -g gitsnap
```

## Usage

```bash
# Snapshot the current directory
npx gitsnap

# Snapshot a specific repo
npx gitsnap /path/to/repo

# Light theme PDF
npx gitsnap --theme light --format pdf

# Skip GitHub API (faster, offline)
npx gitsnap --no-github

# Custom output path
npx gitsnap --output my-card

# With GitHub token (avoids rate limits)
npx gitsnap --token ghp_xxxx
```

## Options

| Flag | Default | Description |
|------|---------|-------------|
| `[repo-path]` | `.` | Path to the git repository |
| `--format` | `png` | Output format: `png`, `pdf`, `svg` |
| `--output` | `gitsnap-output` | Output filename (without extension) |
| `--theme` | `dark` | Card theme: `dark`, `light` |
| `--no-github` | false | Skip GitHub API calls |
| `--width` | `1200` | Card width in pixels |
| `--height` | `630` | Card height in pixels |
| `--token` | - | GitHub personal access token |

## GitHub Token

For public repos, gitsnap works without authentication (60 req/hr). For higher limits or private repos:

```bash
# Environment variable (recommended)
export GITHUB_TOKEN=ghp_xxxx
npx gitsnap

# Or flag
npx gitsnap --token ghp_xxxx
```
